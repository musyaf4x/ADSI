# Changelog

## 3.0.0 — Initial public release

ADSI Agent Kit v3 turns ADSI into a production-oriented audit stack:

- Python CLI: `scan`, `score`, `validate`, `gate`, `collect`, `contrast`, `sarif`, `diff`, `init`, `do`, `mcp`.
- Antigravity-first `/adsi [do] [target]` workflow adapter.
- Agent adapters for Cursor, Claude Code, Codex, Hermes-style runners, OpenClaw, and generic AGENTS.md workflows.
- Static heuristic engine for UI/product slop symptoms.
- Browser artifact collector and optional Node/Playwright package.
- Contrast checker, SARIF reporter, before/after diffing, and MCP-compatible stdio server.
- Tests, examples, reports, and stage-by-stage product audit documentation.
